# bookops-callno

Python library for creation of BPL and NYPl branch call numbers based on bibliographic and order data.

Requires Python 3.8 and up. 

## Version
> 0.1.0

## Installation
Install via pip:
```
python -m pip install git+https://github.com/BookOps-CAT/bookops-callno
```
